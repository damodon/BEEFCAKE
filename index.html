To update your exercises:

  1. Edit the exercises.csv file in your GitHub repository
  2. Commit and push your changes
  3. Refresh this page to see the updates

CSV Format:

Your CSV file should have these columns:
Day,Exercise,Repetitions,Sets,Description,Image

The Image column is optional. Leave it empty if you don't have an image URL.
Use "-" for Repetitions and Sets on rest days.
